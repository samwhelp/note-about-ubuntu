## 注意事項

參考「GitHub / nmakel / [i3-gnome](https://github.com/nmakel/i3-gnome)」提到的

在搭配「i3」時，

要執行下面指令，把「原本的桌面功能」關掉

``` sh
gsettings set org.gnome.gnome-flashback desktop false
gsettings set org.gnome.gnome-flashback root-background true
```
