# sddm


## 安裝指令

``` sh
sudo apt-get install sddm
```


## 切換XDM指令

``` sh
sudo dpkg-reconfigure sddm
```

會出現一個「XDM 選單」提供選擇，請選擇「sddm」，按下「OK」就會切換，

接著登出，或是重新開機，就可以看到使用「sddm」當作登入的介面。
