## Step


### help

run

``` sh
make
```

or run

``` sh
make help
```


### prepare

run

``` sh
make prepare
```


### download

run

``` sh
make download
```


### build

run

``` sh
make build
```


### install

run

``` sh
make install
```


### clean

run

``` sh
make clean
```
