
# Download ISO / Ubuntu


## Prepare

run

``` sh
make prepare
```


## Download

run

``` sh
make download
```


## Mount

run

``` sh
make mount
```

run

``` sh
tree iso-root
```

run

``` sh
tree live-root
```


## Unmount

run

``` sh
make unmount
```


## Extract initrd

run

``` sh
make initrd-extract
```

## Clean

run

``` sh
make clean
```
